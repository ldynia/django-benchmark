## GraphQL

### Queries
```
query {
  allDummy {
    results {
      id
      day
      weekday
      month
      year
      preSeeded
      createdAt
    }
  }
}
```


### Mutations

```
mutation {
  createDummy(newDummy:{ day:1, weekday: "Monday", month: "December", year: 2020}) {
    ok
    dummy {
      id
      day
      weekday
      month
      year
    }
  }
}
```
